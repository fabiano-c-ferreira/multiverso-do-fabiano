Privacy Policy

Last updated: April 17, 2026

Overview

This privacy policy describes how the Personal Automations app ("the App") accesses and uses data from Google and third-party services. The App is a personal automation tool built on a self-hosted n8n instance, used exclusively by its owner (Fabiano Ferreira) to automate personal workflows.

What the App Does

The App connects to various APIs to automate personal tasks, including but not limited to:

  • Reading and updating YouTube video metadata (titles, descriptions, translations, privacy status, and scheduling)
  • Reading content from Notion databases to use as input for automations
  • Any other personal workflow automation the owner may configure in the future

Who Uses This App

This App is for personal use only. It is not distributed to third parties and has no end users other than its owner. No other person has access to this App or its data.

How the App Accesses Google User Data

The App uses Google OAuth 2.0 to authenticate and request access only to the scopes required for its automation tasks. Specifically, the App requests access to:

  • YouTube Data API v3 (https://www.googleapis.com/auth/youtube) — used exclusively to read and update the owner's own YouTube channel video metadata, including titles, descriptions, localized translations, privacy status, and scheduled publish dates.

The App does not access Gmail, Google Drive, Google Contacts, Google Calendar, or any other Google service unless explicitly listed above.

How Google User Data Is Used

Google user data accessed by the App is used solely to perform the automation tasks described above. Specifically:

  • Video titles and descriptions read from Notion are written to the owner's YouTube videos via the YouTube Data API
  • No Google user data is used for advertising, profiling, or any purpose other than the stated automation

How Google User Data Is Stored

  • No Google user data is stored permanently. Data is processed in real time within the owner's self-hosted n8n instance and is not retained after each automation run.
  • Authentication tokens (OAuth 2.0 refresh tokens) are stored securely within the owner's self-hosted n8n instance on a private server, and are never shared with third parties.

How Google User Data Is Shared

Google user data is never shared, sold, or transferred to any third party. All data accessed remains within the owner's private infrastructure.

Third-Party Services

The App may connect to the following third-party services depending on the automation:

  • YouTube Data API v3 (Google LLC) — to read and update video metadata
  • Google OAuth2 — for authentication
  • Notion API — to read content used as input for automations
  • n8n (self-hosted) — as the automation platform

Changes to This Policy

This policy may be updated as new automations are added. The "Last updated" date at the top reflects the most recent revision.

Contact
